# SOCLab

SOCLab documents the implementation of a Security Operations laboratory dedicated to log collection, threat detection and incident investigation.

The objective is to reproduce a realistic monitoring environment capable of detecting common attack techniques while developing practical experience with modern Security Operations workflows.

The laboratory combines log management, detection engineering and threat analysis within a single platform.

---

## Project Objectives

The project focuses on building a small SOC capable of collecting, analysing and correlating security events generated by multiple systems.

The environment progressively integrates detection rules, attack simulations and incident analysis to validate the effectiveness of the monitoring platform.

---

## Technical Scope

Current implementation includes:

- Elastic Stack

## Laboratory Architecture

The target environment consists of:

- **1 Elastic Stack**
- **1 Wazuh Manager**
- **2 Windows Hosts**
- **2 Linux Hosts**
- **1 Kali Linux Attack Machine**
- **25+ Log Sources**
- **20 Detection Rules**
- **10 MITRE ATT&CK Scenarios**
- **6 Security Dashboards**

Each component is progressively documented and validated during implementation.

## Detection Strategy

The laboratory reproduces common attack techniques against Windows and Linux systems.

Security events are collected by Wazuh agents, indexed within Elastic Stack and analysed through custom detection rules.

Each alert is associated with MITRE ATT&CK techniques whenever applicable.

The objective is not only to detect malicious activity but also to understand why the alert was generated and how it can be investigated.

---

## Validation Strategy

The platform is progressively validated using controlled attack scenarios including:

- SSH brute force
- Network discovery
- Port scanning
- Privilege escalation
- Credential access
- Suspicious process execution
- Persistence techniques
- Log integrity verification
- Detection rule validation
- Dashboard verification

Each scenario is documented together with the generated evidence and corresponding detection events.
