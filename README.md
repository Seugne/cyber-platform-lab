# cyber-platform-lab

Laboratoire personnel de cybersécurité orienté Cloud Security, DevSecOps, Kubernetes Security, SIEM Detection Engineering et Secrets Management.

Ce dépôt sert de laboratoire principal pour structurer plusieurs modules techniques :

- 01-terraform-aws : infrastructure AWS sécurisée avec Terraform
- 02-ci-security : pipeline CI/CD sécurisé avec scans automatisés
- 03-k8s-hardening : durcissement Kubernetes et policy-as-code
- 04-detection-lab : supervision sécurité avec Elastic/Wazuh
- 05-vault-secrets : gestion des secrets avec HashiCorp Vault

## Objectif

Construire progressivement une plateforme technique cohérente reproduisant les pratiques utilisées dans un environnement DevSecOps moderne.
